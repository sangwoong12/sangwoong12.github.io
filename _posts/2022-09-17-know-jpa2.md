# JPA를 위한 어노테이션 정리 -2 (다중성)

## @OneToOne
-단방향(unidirectional)
``` java:Member.java
@Entity
public class Member{
    @Id
    private long id;
    
    @OneToOne
    @JoinColumn(name = "locker_id")    
    private Locker locker;
}
```
